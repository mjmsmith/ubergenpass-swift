Master Password

Type your SuperGenPass master password into the two Master Password text fields. The status indicator to the right of the Master Password label changes color as you type. Green indicates that the two passwords match. Red indicates that one or more characters do not match. Yellow indicates that the fields match so far, but one field is shorter than the other.

Secret Password

To set an optional secret password, type it into the two Secret Password fields. The status indicators behave as described above. When the status indicators are green, tap Done to confirm your settings.

Notes

If you've enabled Authenticate With Touch ID: On Launch, UberGenPass stores your master password in the secure iOS Keychain. Otherwise, it stores only a SHA-256 hash of your master password in the Keychain.

If you've set a secret password, it's also stored in the Keychain, AES-256 encrypted using the master password.
