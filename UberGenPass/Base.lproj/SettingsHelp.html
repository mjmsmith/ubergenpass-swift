Master Password

Type your SuperGenPass master password into the Master Password text field. When the text entered matches your current master password, the status indicator to the right of the Master Password label changes to green. Tap Done to continue.

Save Recent Sites

If you enable Save Recent Sites, UberGenPass keeps a list of recent sites for which a password has been generated. These sites are used to populate a suggestion list that updates as you type a site name. Recent sites are stored in the secure iOS Keychain and cleared when you disable the feature.

Use Touch ID

When you relaunch UberGenPass after it's been running in the background, it will ask for your master password again. If you enable Use Touch ID, you can unlock the app with your fingerprint. Note that this setting only applies after you've entered your master password at least once.

Background Timeout

The Background Timeout setting determines how long UberGenPass can be running in the background before asking for your master password again when relaunched. By default, this is set to the most secure option, "immediately". You can optionally suppress this for up to five minutes.
